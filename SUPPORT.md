# Support

## Getting Help

- **Bugs and feature requests:** Open an issue in the relevant repository using the provided templates.
- **Questions and discussions:** Use the Discussions tab on the relevant repository, if enabled.
- **Security issues:** See [SECURITY.md](SECURITY.md) for responsible disclosure.

## What to Include

When asking for help, include:

- What you're trying to do
- What you expected to happen
- What actually happened
- Steps to reproduce (if applicable)
- Relevant versions (OS, language runtime, dependencies)
